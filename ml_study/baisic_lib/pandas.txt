🐼 Pandas란?
Pandas는 표 형태(2차원)의 데이터를 다루는 파이썬 라이브러리야.
엑셀이나 CSV, SQL의 테이블 같은 데이터가 대상이야.

내부적으로는 NumPy 기반이기 때문에 빠르고 강력함.
엑셀보다 100배 빠르고, SQL보다 유연해.


🔑 주요 객체
1. Series: 1차원 벡터 (컬럼 하나)
import pandas as pd
s = pd.Series([10, 20, 30])
pd.Series([30, 35, 40], index=['2015 Sales', '2016 Sales', '2017 Sales'], name='Product A')

2. DataFrame: 2차원 테이블 (행 + 열)
df = pd.DataFrame({
    '이름': ['철수', '영희', '민수'],
    '점수': [90, 85, 78]
})

df = pd.DataFrame(data=None, index=None, columns=None, dtype=None, copy=False)

매개변수    설명
data        생성할 데이터의 본체. 다양한 형태 가능: 딕셔너리, 리스트, 넘파이 배열, 시리즈, 다른 DataFrame 등
index       행 인덱스. 생략하면 기본 인덱스(0부터 시작하는 정수) 사용
columns     열 이름. 생략하면 data에 따라 자동 결정됨
dtype       전체 DataFrame의 데이터 타입. 기본은 자동 추정
copy        True이면 데이터 복사. 원본 변경 방지


📚 핵심 함수/기능 요약

범주        함수/기능                           설명
읽기        read_csv, read_excel, read_sql      다양한 포맷의 파일 로딩
저장        to_csv, to_excel, to_sql            저장
보기        head(), tail(), info(), describe()  미리보기, 요약 통계
인덱싱      df['col'], df.loc[], df.iloc[]      열/행 접근
조건 필터   df[df['col'] > 5]                   필터링
정렬        sort_values, sort_index             정렬
결측값      isnull(), fillna(), dropna()        NaN 처리
그룹화      groupby(), agg()                    그룹 통계
합치기      concat, merge, join                 데이터 병합
빈도 확인   value_counts()                      고유값 개수 계산
인덱스/ 컬럼 이름   rename(), set_index(), reset_index()    이름 변경/인덱스 설정/해제
