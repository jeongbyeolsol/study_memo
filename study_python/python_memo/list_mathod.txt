[ 1. 추가 / 삽입 ]

메서드              설명	                        예시
append(x)           리스트 끝에 x 추가              fruits.append("orange")
insert(i, x)        인덱스 i에 x 삽입               fruits.insert(1, "kiwi")
extend(iterable)    다른 리스트의 모든 요소를 추가  fruits.extend(["melon", "mango"])


[ 2. 삭제 ]

메서드      설명                                    예시
pop([i])    인덱스 i의 요소 삭제 (기본은 마지막)    fruits.pop() / fruits.pop(0)
remove(x)   값이 x인 첫 요소 제거                   fruits.remove("banana")
clear()     모든 요소 제거                          fruits.clear()


[ 3. 정렬 / 반전 ]

메서드              설명                        예시
sort()              리스트 자체를 오름차순 정렬 numbers.sort()
sort(reverse=True)  내림차순 정렬               numbers.sort(reverse=True)
reverse()           요소 순서를 뒤집음          fruits.reverse()


[ 4. 탐색 / 카운트 ]

메서드      설명                예시
index(x)    x의 첫 인덱스 반환  fruits.index("banana")
count(x)    x의 개수 반환       fruits.count("apple")


[ 5. 복사 ]

메서드  설명                예시
copy()  얕은 복사본 반환    new_list = fruits.copy()


[ ✅ 기타: 연산자 지원 ]

연산	설명	    예시
+	    리스트 연결	[1, 2] + [3, 4] → [1, 2, 3, 4]
*	    리스트 반복	[1] * 3 → [1, 1, 1]
in	    포함 여부	2 in [1, 2, 3] → True


[ ✅ 요약 표 ]

기능	메서드
추가	append, insert, extend
삭제	pop, remove, clear
정렬	sort, reverse
탐색	index, count
복사	copy
기타	+, *, in
