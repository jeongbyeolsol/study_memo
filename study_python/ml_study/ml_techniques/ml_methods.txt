🧠 머신러닝의 주요 분류
분류                                    설명                                대표 기법
✅ 지도학습 (Supervised Learning)       정답(label)이 있는 데이터로 학습    분류, 회귀
✅ 비지도학습 (Unsupervised Learning)   정답이 없는 데이터에서 패턴 학습    군집화, 차원 축소
✅ 강화학습 (Reinforcement Learning)    보상을 기반으로 최적의 행동 학습    에이전트, 환경, 보상



1. 지도학습 (Supervised Learning)
✔ 분류 (Classification)
: 정해진 클래스 중 하나로 분류

기법                            설명
K-Nearest Neighbors (KNN)       가까운 이웃 기반 분류
Logistic Regression             확률적 선형 분류
Decision Tree                   의사결정 기반 분기
Random Forest                   다수의 트리를 앙상블
SVM (Support Vector Machine)    분류 경계 최대화
Naive Bayes                     조건부 확률 기반 분류
Neural Networks                 다층 신경망 (딥러닝 포함)


✔ 회귀 (Regression)
: 연속적인 값을 예측

기법                    설명
Linear Regression       직선으로 모델링
Ridge / Lasso           규제를 통한 선형 회귀
SVR                     SVM 기반 회귀
Decision Tree Regressor 분기 기반 예측
Random Forest Regressor 앙상블 기반 예측
Neural Networks         비선형 회귀 가능




2. 비지도학습 (Unsupervised Learning)
✔ 군집화 (Clustering)
: 데이터를 유사한 그룹으로 묶기

기법                    설명
K-Means                 중심 기반 군집화
DBSCAN                  밀도 기반 클러스터링
Hierarchical Clustering 계층적 군집


✔ 차원 축소 (Dimensionality Reduction)
: 고차원 데이터를 저차원으로 압축

기법                설명
PCA (주성분 분석)   분산 보존 축으로 압축
t-SNE               군집 구조 시각화
UMAP                비선형 구조 보존



3. 강화학습 (Reinforcement Learning)
✔ 행동 기반 보상 최적화

개념                    설명
Agent                   학습하는 주체
Environment             행동이 일어나는 공간
Reward                  행동에 대한 보상
Q-Learning              행동-보상 값을 학습
Policy Gradient         확률적인 정책을 최적화
Deep Q-Network (DQN)    딥러닝을 적용한 Q-Learning



⚙️ 기타 실전에서 자주 쓰이는 개념

개념                설명
하이퍼파라미터 튜닝 GridSearchCV, RandomizedSearchCV
교차 검증           cross_val_score()
특성 선택           SelectKBest, RFE
표준화              StandardScaler, MinMaxScaler
오버샘플링          SMOTE
불균형 대응         class_weight, undersampling
